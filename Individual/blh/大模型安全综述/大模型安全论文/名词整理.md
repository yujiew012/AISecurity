# 大模型安全核心名词整理
## 一、核心基础名词
1. 大语言模型（LLM）：基于海量文本训练、数十亿至千亿级参数的深度学习模型，具备强自然语言理解与生成能力，应用于文本交互、创作、分析等，代表有 ChatGPT、文心一言。
2. 预训练模型（PLM）：在大规模无标注数据完成初始训练的基础模型，具备基础语言能力，可通过微调适配特定领域与任务。
3. Transformer 架构：大语言模型核心技术框架，凭借并行计算与多头自注意力机制高效处理长文本、捕捉深层语义，是 BERT、GPT 等基础架构。
4. 编码器 / 解码器模型：大模型两大核心类型。编码器（如 BERT）侧重语义理解，用于分类、实体识别；解码器（如 GPT）侧重文本生成，用于续写、创作。
5. 开源 / 闭源模型：开源模型（Llama、ChatGLM）公开源码与参数，支持二次开发；闭源模型（ChatGPT、Claude 3）不公开核心内容，仅通过 API 调用。
6. 安全垂直领域大模型：针对网络安全、数据安全、公共安全等场景定制的专业化大模型，可精准解决领域专业问题，如 360 安全大模型、奇安信 Q-GPT。
7. 自研安全大模型：企业依托自身技术、安全数据与行业经验独立研发的安全专用大模型，场景针对性与专业适配性极强。
8. 成熟大模型赋能安全：利用通义千问、ChatGPT 等通用大模型，通过预训练 + 微调适配安全任务，快速具备安全领域能力。
9. 大模型安全平台：覆盖大模型训练、精调、推理、部署、运营全生命周期的综合安全保障平台，提供风险检测、漏洞防护、数据加密、内容审核等一站式服务。
10. 微调：以通用大模型为基础，使用安全领域专属数据（漏洞案例、攻击日志等）进一步训练，使模型掌握专业知识、适配安全场景。

## 二、安全风险与伦理名词
1. 隐私泄露：模型在训练 / 使用中因缺陷、漏洞或不当操作，导致敏感数据、用户交互信息外泄。
2. 模型偏见与歧视：受训练数据偏见影响，模型对特定群体产生不公平表述或决策，违背公平性。
3. 模型幻觉：生成看似合理但与客观事实不符的内容，包括编造知识、错误关联信息。
4. 可信度问题：因模型能力缺陷、数据偏差，生成虚假、误导内容或恶意代码，带来安全风险。
5. 鲁棒性不足：面对输入扰动或恶意攻击时，模型性能大幅下降，无法稳定输出安全准确结果。
6. 安全对齐偏差：模型输出与人类安全价值观、伦理及规则不一致，存在有害内容生成或被恶意利用风险。

## 三、攻击手段相关名词
1. 提示词注入（PI）：构造恶意提示诱导模型绕过安全规则，生成有害内容或泄露信息，分直接注入与间接注入。
2. 数据投毒：在训练数据中混入错误、有害或带后门样本，破坏模型性能、植入恶意功能。
3. 后门攻击：在模型训练 / 部署阶段植入隐藏触发条件，满足条件时输出有害内容，隐蔽性极强。
4. 自动化攻击：利用工具批量生成恶意提示或攻击样本，高效挖掘模型漏洞。
5. 模型窃取（Model Leeching）：通过多次查询提取模型知识与参数，复刻功能相近模型。
6. 社会工程学攻击：借助大模型模拟人类交流，以钓鱼、假冒身份诱导用户泄露信息或执行危险操作。

## 四、安全防御相关名词
1. 对抗训练：训练中引入恶意样本，让模型学习攻击特征，提升识别与抵御能力。
2. 模型加固：调整结构、优化配置、完善流程，提升抗攻击能力，修复后门、增强鲁棒性。
3. 数据隔离：隔离用户数据与模型训练、推理过程，限制必要数据交互，防范数据泄露。
4. 水印技术：在生成内容中植入可验证标记，实现 AI 内容溯源与知识产权保护。
5. 输入 / 输出控制：对输入进行安全过滤、对输出做合规审查，降低安全风险。
6. 安全评估框架：标准化检测模型漏洞、偏见、可信度与抗攻击能力的准则与工具。
7. 智能攻击对抗：模拟攻击者视角生成样本与场景，检测模型安全弱点，优化防御，如蚁鉴 2.0。
8. 反馈学习：模型依据用户反馈与应用效果持续优化，提升安全检测与防范准确性。
9. 安全对齐策略：使模型行为与决策符合人类安全规则，避免生成有害内容或被滥用。
10. 全生命周期安全：对大模型训练、精调、推理、部署、运营各环节均实施安全防护。

## 五、应用场景相关名词
1. 网络安全应用：用于渗透测试、传统 / 新型蜜罐、网络威胁检测（病毒、DDoS 等）。
2. 物理安全应用：保障芯片、智能家居、物理设备等硬件与实体环境安全。
3. 信息安全应用：代码生成与漏洞修复、智能合约审计、电信诈骗防范、静态代码检测。
4. 服务配置错误识别：分析系统配置文件，检测并纠正配置不当引发的漏洞。
5. 安全风险发现：主动识别系统漏洞、潜在威胁，如钓鱼邮件、恶意代码。
6. 数据泄露管理：监控数据输入输出与流转，预警并处理敏感信息泄露风险。
7. 安全运营平台：集成大模型能力，实现安全监控、攻击预警、应急响应，如恒脑・安全垂域大模型运营平台。
8. 社会工程学防御：识别钓鱼、假冒身份等攻击，提醒用户防范，降低诈骗成功率。
