# Individual — 个人学习空间

本目录为**个人学习与研究空间**，每位成员可在此建立自己的子目录。

## 目录结构

```
Individual/
└── <name>/          # 以你的名字或昵称命名
    └── <主题>/      # 按主题分类（如：大模型安全综述、对抗样本等）
        └── ...      # 论文、笔记、心得等
```

## 使用说明

- **子目录命名**：在 `Individual/` 下创建以自己名字或昵称命名的文件夹（如 `tyx/`、`blh/`）
- **内容组织**：在个人目录内按**主题**建文件夹，存放该主题下的论文、阅读笔记与心得
- **用途**：可作为个人暂存区与知识沉淀空间，便于后续整理或迁移到 `Papers/`、`Tools/` 等
